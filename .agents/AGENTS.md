# Project Custom Rules for DIGIWILL

- **Mobile First & Responsive Design by Default**: Always build, layout, and style components prioritizing mobile viewports. Ensure that select boxes, tables, forms, and buttons wrap and scale automatically using Tailwind CSS responsive classes (e.g., `flex-col sm:flex-row`, `w-full max-w-full`, and `min-w-0` on flex items) so that the application never requires horizontal scrolling or overflows on cellular phone screens.
